# o2n
## Running production build

In project root directory run:
```
docker-compose up
```

## Frontend
### Project setup

Enter `frontend` directory

```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn serve
```

#### Compiles and minifies for production
```
yarn build
```

#### Run your unit tests
```
yarn test:unit
```

#### Lints and fixes files
```
yarn lint
```

## Backend

Enter `backend` directory

Run
```
yarn install
```
and then

```
yarn start
```
